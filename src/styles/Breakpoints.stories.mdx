# Breakpoints

Breakpoints for responive layout:

<table>
  <tr>
    <th> Extra Small (xs)</th>
    <th> Default</th>
  </tr>
  <tr>
    <th> Small (ms)</th>
    <th> >= {BreakpointSizes.sm}px</th>
  </tr>
  <tr>
    <th> Mediun (md)</th>
    <th> >= {BreakpointSizes.md}px</th>
  </tr>
  <tr>
    <th> Large (ms)</th>
    <th> >= {BreakpointSizes.lg}px</th>
  </tr>
  <tr>
    <th> Extra Large (ms)</th>
    <th> >= {BreakpointSizes.xl}px</th>
  </tr>
</table>

Using in styled component:

```js
import {breakAt, BreakpointSizes} from Breakpoints;

const Title = styled.h1`
  font-size: 1rem;
  ${breakAt(BreakpointSizes.md)}{
    font-size: 2rem;
    }
`;
```
